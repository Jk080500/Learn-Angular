What is Angular?

- Javascript framework for web development
- Single Page Application ( means No Reload on menu changes)
- Developed and Maintained By Google
- Node should be installed to use the angular
- Angular follows the MVC Architecture
- Angular js and Angular are different

Why should we learn Angular?

- Fast Loading and Development
- Angular Provide alots of features
  - Validation Library
  - Routing
  - Component bases structure
- Large Community
